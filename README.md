## üöÄ Live Hosting Guide (GitHub Pages)

This project is pre-configured for **GitHub Actions** deployment.

### Step 1: Push to GitHub
Ensure your code is pushed to your repository: `https://github.com/Kelvin-wk/Susan-Market`.

### Step 2: Set the API Key
1. Go to **Settings > Secrets and variables > Actions**.
2. Add a **New repository secret**.
3. **Name**: `API_KEY`
4. **Value**: (Your Gemini API Key from Google AI Studio).

### Step 3: Configure Deployment
1. Go to **Settings > Pages**.
2. Change **Source** to **GitHub Actions**.
3. Push any small change to your `main` branch (e.g., editing this README) to trigger the first build.

### Step 4: Access Your Site
Your site will be available at:
`https://kelvin-wk.github.io/Susan-Market/`
